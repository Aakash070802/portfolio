## 🧠 About the Project

This portfolio is not just a static page — it's designed to create a **smooth, animated, and engaging user experience**.

It includes:

- Intro animation before loading main content
- Custom cursor interaction
- Particle-based background effects
- Smooth transitions using Framer Motion
- Fully responsive design

## 🔐 Environment Variables

Create a `.env` file in root:

```
VITE_SERVICE_ID=your_service_id
VITE_TEMPLATE_ID=your_template_id
VITE_PUBLIC_KEY=your_public_key
```

⚠️ App will crash if these are missing (intentionally enforced in config).
